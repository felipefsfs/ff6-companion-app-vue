# ff6-companion-app
Companion App for FF6, any version. Tracks Exp, hit optimum Esper lvl Bonus.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
